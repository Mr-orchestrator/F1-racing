# Playwright Debug & Session Replay

The Playwright config (`playwright.config.js`) is already set to record full traces and video on every
run (`trace: 'on'`, `screenshot: 'on'`, `video: 'on'`) — so every test run is a replayable session.

## Debug a test live

```bash
# UI mode — pick a test, watch it run, time-travel through every step
npx playwright test --ui

# Inspector — pause at the first action, step through
PWDEBUG=1 npx playwright test tests/full-funnel-validator.spec.js

# Headed (see the browser)
npx playwright test tests/full-funnel-validator.spec.js --headed

# Single test, single project
npx playwright test tests/full-funnel-validator.spec.js --project=chromium --reporter=list
```

## Replay a previous session

After any run, Playwright writes per-test artifacts under `test-results/`:

```
test-results/
  <spec-name>--<test-id>--chromium/
    ├── trace.zip          ← full DOM + network + console replay
    ├── video.webm         ← screen recording of the run
    ├── test-failed-1.png  ← screenshots on failure
    └── error-context.md   ← what was on screen when it failed
```

Open a trace in the trace viewer (full time-travel, network, console, DOM):

```bash
npx playwright show-trace test-results/<...>/trace.zip
```

Or watch the HTML report (auto-generated by every run):

```bash
npx playwright show-report
```

## Validator reports

`tests/full-funnel-validator.spec.js` writes a structured run report to a timestamped folder:

```
validation-reports/
  2026-06-29-12-15-13/
    ├── cross-stream-report.xlsx   ← Summary | Events | Parameters | GA4 Network | Adobe Network | Console Errors
    └── cross-stream-report.json   ← machine-readable
```

The Excel + JSON are also attached to the Playwright HTML report for that run, so the trace replay
and the cross-stream data sit side-by-side.

## Common debug snippets

```bash
# Just see what each test does — no actions
npx playwright test --list

# Re-run only failed tests from the last session
npx playwright test --last-failed

# Run against prod instead of localhost
BASE_URL=https://racing-f1-rho.vercel.app npx playwright test tests/full-funnel-validator.spec.js --project=chromium
```
